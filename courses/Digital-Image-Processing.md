# 课程信息

数字图像处理

课程开设时间 ：学年上学期

推荐选课年级 ：大二及以上

课程号 : 304052020

## 课程内容
- 数字图像处理绪论
- 图像知识和运算
- 图像增强处理理论及算法
- 图像压缩编码
- 图像分割基本方法



## 前置课程
> 基础工具
- Python
- MATLAB
